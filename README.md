# KharidoBecho.com

A Website for buying and selling old cars.


## Backend 
Spring BOOT
## Frontend
AngularJS
## Database
MySQL
## ORM 
JPA/ Hibernate


## Commands to run
npm install -g @angular/cli </br>
ng new myproject </br>
cd myproject </br>
npm install bootstrap@3 jquery --save </br>
ng serve </br>
